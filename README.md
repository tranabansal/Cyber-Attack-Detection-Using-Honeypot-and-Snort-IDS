Overview:

This project focuses on detecting cyber attacks on web applications using a Honeypot and Snort Intrusion Detection System (IDS).

The system captures attacker activities, logs malicious requests, and provides centralized log management for security monitoring.

Technologies Used:

Kali Linux |
Snort IDS |
Pentbox Honeypot |
Nginx |
PHP |
MySQL |
Features |
Honeypot Deployment |
Intrusion Detection |
Attack Logging |
Centralized Log Management |
Security Monitoring Dashboard
